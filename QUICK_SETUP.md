# QUICK SETUP GUIDE - Goodness Omogo School Website

## ⚡ Quick Start (5 Minutes)

### Step 1: Install XAMPP/WAMP
- Download XAMPP: https://www.apachefriends.org/
- Install and start Apache + MySQL services

### Step 2: Setup Database
1. Open browser: http://localhost/phpmyadmin
2. Click "New" to create database
3. Name it: `goodness_omogo_db`
4. Click "Import" tab
5. Choose file: `database_schema.sql`
6. Click "Go" button

### Step 3: Deploy Files
1. Copy entire `goodness_omogo_school` folder to:
   - Windows: `C:\xampp\htdocs\`
   - Mac: `/Applications/MAMP/htdocs/`
2. That's it!

### Step 4: Access Website
Open in browser:
- Homepage: http://localhost/goodness_omogo_school/
- Result Checker: http://localhost/goodness_omogo_school/result-checker/
- Admin Portal: http://localhost/goodness_omogo_school/admin/login.php

### Step 5: Login to Admin
- Username: `admin`
- Password: `admin123`
